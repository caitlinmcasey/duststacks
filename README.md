# duststacks
Hosts data tables from Casey et al. (2026) manuscript: "Dust in the Average Galaxy: Attenuation, Emission, and Opacity from 0&lt;z&lt;7"

** casey26_duststacks_submmfluxdensities.fits ** -- A table of measured (sub)mm flux densities for the stacked JWST galaxy population described in the paper.  All flux densities are indicated in observed mJy.  (Table 1)

** casey26_duststacks_derivedphysical.fits ** -- A table of derived physical characteristics from SED fitting (Table 2)
